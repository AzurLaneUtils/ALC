# AzurLaneCalculator

我们发现，现有的与碧蓝航线相关的计算器大多是有关CD的计算。而对于其他的方面，例如战斗内的buff计算没有很好的支持。

该项目的目的正是为了解决以上问题，实现一个功能支持较全的计算器。
